Gantt Chart

Our Gantt chart is divided into the major phases our team expected to have.

Phase 1 focuses on creating a base map with default interest point locations. Phase 2 focuses on the user login and accout creation page with a built in security system. Phase 3 focuses on the feature for users to add in custom locations to the map. Phase 4 focuses on features we consider to be features that are nice to have. There is a misc table with tasks that do not fit into a specific phase but have a timline duration for the tasks in the table.
